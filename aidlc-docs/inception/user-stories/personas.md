# User Personas

## Persona 1: Lê — PC Builder / Gamer

| Attribute | Detail |
|---|---|
| **Name** | Lê |
| **Role** | Primary user (personal tool) |
| **Technical Level** | Intermediate — biết cài driver, dùng command line cơ bản, nhưng không phải developer |
| **Motivation** | Muốn kiểm soát màu RGB trên GPU Colorful iGame RTX 5070 vì phần mềm chính hãng không hoạt động |
| **Goals** | Set màu yêu thích, chạy effects đẹp, đổi màu theo mood/game, tắt LED khi cần |
| **Frustrations** | iGame Center và OpenRGB không detect được card; LED bị kẹt ở màu không mong muốn |
| **Environment** | Windows 11, single monitor setup, card duy nhất trong case |
| **Usage Pattern** | Mở app khi muốn đổi màu, sau đó minimize to tray; thỉnh thoảng dùng dynamic color |

Vì đây là personal tool (chỉ 1 user), persona đơn giản. Tuy nhiên, stories vẫn follow best practices để đảm bảo UX rõ ràng và testable.
